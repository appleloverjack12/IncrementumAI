# IncrementumAI

Multi-agent AI system for company expansion using ElizaOS.

## Agents

1. **Strategy Coordinator** - Orchestrates all agents
2. **Market Research** - Analyzes markets and competitors
3. **Outreach** - Crafts personalized communications
4. **Content** - Creates marketing materials
5. **Lead Qualifier** - Scores and prioritizes leads

## Deployment

Deployed on Railway with Discord integration.
